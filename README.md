# WEDE5020
Web development Project

## Project Title
Apple Electronic Company

## Overview of Project
This project contains different pages of the Apple Company and their branches the Istore. It contains multiple webpages that contain different information about the company.

## Features
interactive map with clickable markers
interactive product gallery using lightbox
interactive pictures using lightbox
Contact form with errors and immediate feedback
meta descriptions and keywords
navigation bar adjusts for mobile,tablet and desktop computer
interactive phone numbers when pressed directly go to phone icon

## Technological and languages used
Css
Javascript
Html
lightbox
leaflet.js
openstreetmap
Github

## Objectives
To create a proffessional and responsive website
Introduce Apple products and services 
use javascript elements
deploying this website using Github
